# Myself_python_study


## 仓库结构
该仓库作为个人学习python中的一些记录和思考，以`week`的形式来进行记录：注意这里的week并不表示现实中连续的一周时间，而是`理想`中一周学习时间内应该实现的项目和知识对应的任务量,因而日期范围并不固定。其次，我将以问题形式作为基本导向，给出一些基本问题和测试内容，目的是促进思考并检验个人知识掌握情况。

sourse目录下对应学习过程中使用到的一些有用的资源。
